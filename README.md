# Waypoint

A lightweight personal travel itinerary tool.

## Features
- Timeline-based trip overview
- Natural language segment entry (GPT-4o)
- Email parsing (coming soon)
- Clean, fast web UI

## Stack
- **Backend:** FastAPI + SQLAlchemy + SQLite
- **Frontend:** Vanilla JS, served as static files
- **AI:** OpenAI GPT-4o
- **Infra:** Ubuntu 24.04, Nginx, Let's Encrypt
